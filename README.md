App Info
------------------------------------------------------
- 스마트폰을 소음 측정하고 싶은 장소에 가서 장소명을 등록한 후 측정을 시작(스마트폰 마이크로 소음 측정) <br>
- 주기적으로 웹서버에 장소와 데시벨 구간별로 나눈 시끄러운 정도를 실시간으로 업로드 <br>
- 해당 어플에 접속하면 실시간으로 어떤 장소에 얼마나 소음이 있는지 확인 가능 <br>
